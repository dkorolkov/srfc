# Тестовое задание

3 пользователя, 3 пункта меню и 3 таблицы

## Развёртывание приложения

`bash install.sh`

## Запуск приложения

`bash run.sh`

## Работа с приложением

Открыть в браузере страницу

`http://localhost:8000/menu/`

Войти под одним из пользователей:

- user1:password1
- user2:password2
- user3:password3
